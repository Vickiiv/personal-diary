# Personal Diary

Ein einfaches Tagebuch-Projekt zum Üben von React, State Management und lokaler Datenspeicherung.

🔗 **Live-Demo:** https://personal-diary-0uu9.onrender.com

## Über das Projekt

Dieses Projekt ist im Rahmen meiner Weiterbildung als Übung entstanden. Ziel war es, die Grundlagen von React-Komponenten, State Management und einem responsiven UI-Design praktisch anzuwenden.

Hinweis: Als Lernprojekt gibt es noch kleinere Bugs, die ich in Zukunft beheben möchte.

## Tech-Stack

- JavaScript
- React (Vite)
- TailwindCSS
- localStorage (zur lokalen Datenspeicherung)

## Lokal starten

\`\`\`bash
npm install
npm run dev
\`\`\`
